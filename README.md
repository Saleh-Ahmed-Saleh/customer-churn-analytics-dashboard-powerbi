📊 Customer Churn Analysis Dashboard (Power BI)
📁 Project Overview

This project presents a professional Customer Churn Analytics Dashboard built using Power BI and real-world banking data.

It demonstrates a complete end-to-end data analytics workflow, from raw data transformation to advanced insights generation.

The dashboard enables businesses to:

Monitor customer churn and retention
Identify high-risk customer segments
Analyze behavioral and demographic patterns
Support data-driven decision-making

🧱 Data Model

The project follows a Star Schema Model for performance optimization and scalability.

🔸 Fact Table:
Customer Churn Data (CustomerID, Balance, Credit Score, Tenure, etc.)
🔸 Dimension Tables:
Age Range
Balance Range
Credit Score Range
Tenure Range
Geography
🔸 Relationships:
One-to-Many relationships between dimension tables and fact table
Optimized for fast filtering and slicing
⚙️ Data Preparation & Modeling
Cleaned and transformed raw data using Power Query
Created custom grouping dimensions:
Age Groups
Balance Segments
Credit Score Categories
Built efficient relationships between tables
Improved performance using proper modeling techniques
